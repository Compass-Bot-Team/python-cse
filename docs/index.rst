Welcome to Python-CSE!
=======================

Python-CSE is a small, asynchronous wrapper around the Google Programmable Search Engine.

**Features**

- Completely asynchronous API
- 100% API coverage
- Fully typed code
- Image searching
- SafeSearch toggleable
